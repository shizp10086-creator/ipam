## API 集成

使用的 API 接口（已在 `frontend/src/api/ip.js` 中定义）:
- ✅ `getIPs()` - 获取 IP 列表
- ✅ `getIP(id)` - 获取 IP 详情
- ✅ `allocateIP(data)` - 分配 IP
- ✅ `releaseIP(data)` - 回收 IP
- ✅ `updateIP(id, data)` - 更新 IP（用于保留/取消保留）
- ✅ `checkConflict(data)` - 检查冲突

使用的其他 API:
- ✅ `getSegments()` - 获取网段列表
- ✅ `getDevices()` - 获取设备列表
- ✅ `getLogs()` - 获取操作日志
