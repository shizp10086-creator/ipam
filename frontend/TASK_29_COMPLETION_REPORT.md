## 技术实现细节

### 使用的技术栈
- **Vue 3**: Composition API
- **Element Plus**: UI 组件库
- **ECharts 5**: 数据可视化图表库
- **Axios**: HTTP 请求

### 核心功能
1. **关键指标卡片**
   - 使用 el-card 和 el-row/el-col 布局
   - 悬停效果（阴影和位移）
   - 图标和数值展示
   - 响应式设计

2. **图表组件**
   - 网段使用率柱状图（带点击交互）
   - IP 状态分布环形饼图
   - 设备数量趋势折线图（带面积填充）
   - 自适应窗口大小变化

3. **数据刷新机制**
   - 自动定时刷新（5 分钟）
   - 手动刷新按钮
   - 加载状态提示
   - 错误处理和用户提示

4. **时间范围控制**
   - 日期范围选择器
   - 快速时间范围切换（7/30/90 天）
   - 动态更新图表数据

### API 集成
- `GET /api/v1/dashboard/stats` - 获取统计数据
- `GET /api/v1/dashboard/charts?days={days}` - 获取图表数据
- `GET /api/v1/logs` - 获取最近操作日志
- `GET /api/v1/alerts` - 获取未解决告警

### 响应式设计
- 支持桌面端（≥768px）
- 支持移动端（<768px）
- 自适应布局调整
- 图表高度自适应
